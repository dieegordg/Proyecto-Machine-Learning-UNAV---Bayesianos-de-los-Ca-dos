# DECISIONES_notebook.md

## Objetivo

Registrar las decisiones técnicas tomadas durante la construcción del notebook de datos, para facilitar después la redacción del informe parcial y final.

## Decisión 1 — Reinicio del notebook de datos

Se decide crear un notebook nuevo desde cero porque el notebook anterior no sigue un flujo suficientemente claro para la fase actual del proyecto.

El nuevo notebook se centrará primero en responder:

> ¿Qué se puede analizar realmente con estos documentos?

## Decisión 2 — Evaluación de fuentes

Se evaluarán tres fuentes:

1. Buscador RTVE 23-F.
2. Lista íntegra de documentos RTVE.
3. Página oficial de La Moncloa.

La prioridad inicial será RTVE, porque es el dataset principal indicado en el enunciado. La Moncloa se usará como fuente oficial de contraste y validación de cobertura.
